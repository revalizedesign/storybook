# Usage

## AGENTS.md / CLAUDE.md

`CLAUDE.md` at the project root defines rules for AI agents working in this repo. It is automatically loaded by Claude Code at the start of every session, ensuring consistent behavior across conversations. Any developer or agent contributing to this project should read it first.
